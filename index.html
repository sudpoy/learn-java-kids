What is Java?

Java is a programming language — a way to give instructions to a computer. When you write Java code, the computer reads it and does exactly what you tell it to!


Java was created in 1995 and is used by billions of devices. Your first Java program says "Hello, World!" to the screen.

Your First Program

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
📟 Output: Hello, World!

🔑 Key Words Explained

  • class — A blueprint for your program (like a recipe card)
  • main — Where Java starts running your code
  • System.out.println() — Prints text to the screen
📦

Variables & Data Types

A variable is like a labeled box where you store information. You give it a name so you can use it later!

Common Data Types

// int — whole numbers
int age = 10;
int score = 250;

// double — numbers with decimals
double height = 4.5;

// String — text (always in "quotes")
String name = "Alex";

// boolean — true or false
boolean isHappy = true;

Doing Math with Variables

int apples = 5;
int oranges = 3;
int total = apples + oranges;
System.out.println("Total fruit: " + total);
📟 Output: Total fruit: 8

Loops

A loop repeats code many times. Instead of writing the same line 100 times, loops do it for you automatically!

For Loop — Count from 1 to 5

for (int i = 1; i <= 5; i++) {
    System.out.println("Count: " + i);
}
📟 Count: 1
📟 Count: 2
📟 Count: 3
📟 Count: 4
📟 Count: 5

While Loop — Keep going until done

int cookies = 5;
while (cookies > 0) {
    System.out.println("Eating cookie #" + cookies);
    cookies--;  // remove one cookie
}
System.out.println("All cookies eaten!");

If / Else — Making Decisions

Programs need to make decisions. if/else checks a condition and does different things based on whether it's true or false.

Temperature Check

int temperature = 30;

if (temperature > 25) {
    System.out.println("It's hot! Go swimming! 🏊");
} else if (temperature > 15) {
    System.out.println("Nice weather! Go outside! 🌤️");
} else {
    System.out.println("It's cold! Wear a jacket! 🧥");
}
📟 It's hot! Go swimming! 🏊

Methods — Reusable Code

A method is a named block of code you can call whenever you need it. Think of it like a recipe — write it once, cook it anytime!

Greet Method

// Define the method
static void greet(String name) {
    System.out.println("Hello, " + name + "! Welcome to Java! 🎉");
}

// Call the method
public static void main(String[] args) {
    greet("Alex");
    greet("Sam");
    greet("Jordan");
}
📟 Hello, Alex! Welcome to Java! 🎉
📟 Hello, Sam! Welcome to Java! 🎉
📟 Hello, Jordan! Welcome to Java! 🎉

Method that Returns a Value

static int add(int a, int b) {
    return a + b;
}

// In main:
int result = add(4, 7);
System.out.println("4 + 7 = " + result);
📟 4 + 7 = 11
